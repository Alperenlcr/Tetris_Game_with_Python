# Tetris_Game_with_Python

## How to Play
| Command      | Key           |
|--------------|---------------|
| Move left    | A             |
| Move right   | D             |
| Rotate       | R             |
| Hard drop    | S             |
| Pause game   | P             |
| Quit game    | Q             |

## How to Run
- First install neccessary modules
```
 $ pip install getch-1.0
 $ pip install emoji-2.2.0
```
- Launch game
```
 $ python tetris.py
```
